## App Setting
- .env file (for setting maximum count of course)
REACT_APP_MAX_COUNT = 5

## App Install & Run
- App install
    npm install
- App start
    npm run start
- App build
    npm run build

## App Detail
This app was built using CRA, Eslint and TailwindCSS.
React-Redux was used for storing personal detail and course detail.
